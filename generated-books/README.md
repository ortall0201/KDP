# Generated Books

This folder contains AI-generated book manuscripts from the KDP Content Production workflow.

## File Format

Files are saved as: `{timestamp}_{niche_name}_manuscript.txt`

Example: `20260103-220000_Keto_Diet_for_Beginners_manuscript.txt`

## Usage

1. Review content for quality
2. Edit as needed
3. Format for KDP (convert to .docx if needed)

## Git Management

These files are tracked in git. You can:
- **Keep them**: Commit to version control
- **Ignore them**: Add `*.txt` to .gitignore if files are too large

Average file size: ~50-100 KB per book
