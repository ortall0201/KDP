# Niche Research Results

This folder contains profitable niche opportunities discovered by the automated research workflow.

## Files

Each file is named: `niche_opportunities_YYYY-MM-DD.md`

## Format

Each opportunity includes:
- **Niche name**
- **ROI percentage** (400%+ minimum)
- **Monthly revenue estimate**
- **Optimal pricing**
- **Competition level**
- **Keywords** (7 suggested)
- **Target audience**
- **Book angle**

## Usage

1. Review opportunities in this folder
2. Pick the best one
3. Use that niche in Workflow 2 (Content Production)
4. Generate your book!

## Workflow

Workflow 1 runs **every 3 days** automatically and commits new opportunities here.

Check this folder regularly for fresh niche ideas!
