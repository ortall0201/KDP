## üÜò TROUBLESHOOTING

### Common Issues

**"Workflow not running"**
- Check workflow is activated (toggle in n8n)
- Verify credentials are connected
- Check execution history for errors

**"OpenAI quota exceeded"**
- Upgrade OpenAI plan
- Reduce requests per day
- Use lower temperature/fewer tokens

**"Google Sheets not updating"**
- Verify Sheet ID is correct
- Check OAuth permissions
- Ensure sheet tabs match exactly

**"Book content is low quality"**
- Review and edit AI output always
- Use better prompts
- Mix AI tools (Claude + GPT-4)
- Add human examples

**"No sales after 2 weeks"**
- Improve book cover
- Rewrite description
- Lower price temporarily
- Get initial reviews (friends/family)
- Check preview quality
