# Generated Covers

This folder contains AI-generated book covers from the KDP Cover Design workflow.

## File Format

Files are saved as: `cover_{timestamp}_{niche_name}.png`

Example: `cover_20260103-220000_Keto_Diet_for_Beginners.png`

## Specifications

- Format: PNG
- Size: 1024x1792 pixels (portrait)
- Quality: HD
- Note: Text overlay should be added in Canva/Photoshop before KDP upload

## KDP Requirements

For KDP upload, covers must be:
- **Minimum**: 1000 pixels on shortest side
- **Ideal**: 1600x2560 pixels (2.5:4 ratio)
- **Our output**: 1024x1792 (close to ideal ratio)

## Git Management

These files are tracked in git. You can:
- **Keep them**: Useful for tracking cover iterations
- **Ignore them**: Add `*.png` to .gitignore if repo size becomes large

Average file size: ~2-5 MB per cover
